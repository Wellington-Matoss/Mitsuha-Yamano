# 🦊 BABYMETAL Fan Site

Site de fã dedicado ao grupo japonês **BABYMETAL**, combinando a estética do heavy metal com elementos da cultura pop japonesa (kawaii metal).

---

## 📁 Estrutura do Projeto

```
src/
├── index.html          # Página inicial (sobre a banda e discografia)
├── integrantes.html    # Página de integrantes (atuais e ex-membros)
├── css/
│   └── main.css        # Folha de estilos principal
└── js/
    └── main.js         # JavaScript (reservado para futuras funcionalidades)
```

---

## 🎸 Páginas

### `index.html` — Página Inicial

- Apresentação da banda
- Discografia principal
- Link para a página de integrantes

### `integrantes.html` — Integrantes

- **Su-Metal** (Suzuka Nakamoto) — Vocalista principal
- **Moametal** (Moa Kikuchi) — Vocalista & Dançarina
- **Momometal** (Momoko Okazaki) — Vocalista & Dançarina _(membro oficial desde 2023)_
- **Yuimetal** (Yui Mizuno) — Ex-integrante _(saiu em 2018)_

## 🎨 Design

O tema visual é inspirado na identidade do BABYMETAL:

- **Paleta:** preto profundo, vermelho (`#cc0000`) e dourado (`#ffd700`)
- **Tipografia:** [Cinzel Decorative](https://fonts.google.com/specimen/Cinzel+Decorative) para títulos · [Noto Sans JP](https://fonts.google.com/noto/specimen/Noto+Sans+JP) para corpo
- **Layout:** responsivo com CSS Grid e `clamp()` para tipografia fluida
- **Acessibilidade:** suporte a `prefers-reduced-motion`, foco visível, atributos ARIA
